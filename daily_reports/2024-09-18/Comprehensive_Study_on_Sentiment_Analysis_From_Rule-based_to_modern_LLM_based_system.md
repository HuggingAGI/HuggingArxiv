# 情感分析研究全景：从传统规则系统到现代 LLM 驱动系统
发布时间：2024年09月16日


> Comprehensive Study on Sentiment Analysis: From Rule-based to modern LLM based system
>
> 本文深入探讨了 AI 和 LLM 背景下的情感分析，从传统规则方法到现代深度学习技术，全面回顾了其发展历程。重点分析了从词典和模式识别到机器学习和深度学习模型的转变，并探讨了处理双语文本、识别讽刺和消除偏见等关键挑战。通过总结当前技术并展望未来趋势，本文旨在为 AI 和 LLM 背景下的情感分析提供全面洞察。
>
> https://arxiv.org/abs/2409.09989
