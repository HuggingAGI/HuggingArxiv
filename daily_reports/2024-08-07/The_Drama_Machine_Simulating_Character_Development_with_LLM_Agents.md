# 戏剧机器：借助 LLM 代理模拟角色成长
发布时间：2024年08月03日

`Agent应用`
> The Drama Machine: Simulating Character Development with LLM Agents
>
> 摘要：本文通过多个大型语言模型（LLM）代理，探索在戏剧性场景中模拟复杂动态角色的方法。我们提出的“戏剧机器”框架，协调了扮演“自我”与“超我”角色的LLM代理间的互动，使主体间对话与主体内内心独白并行展开。我们将其应用于采访和侦探故事两个场景，对比了超我影响下的角色发展。初步结果显示，多代理方法能创造出更细腻、适应性强的叙事，随对话推进而演变。文章还探讨了基于LLM的角色扮演与发展的多种形式，及其对AI主体性概念化的影响。最后，我们思考了这种方法如何为AI模拟中的内心冲突和社会表演性提供了新的视角。
>
> https://arxiv.org//pdf/2408.01725
