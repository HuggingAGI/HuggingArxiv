# 通过检索增强生成技术，优化可持续城市旅行的旅游推荐系统
发布时间：2024年09月26日

`RAG`
> Enhancing Tourism Recommender Systems for Sustainable City Trips Using Retrieval-Augmented Generation
>
> 传统的旅游推荐系统 (TRS) 主要关注个性化建议，往往忽视了可持续性目标。随着环保和社区利益的重要性日益凸显，将可持续性融入 TRS 变得至关重要。本文提出了一种创新方法，利用大型语言模型 (LLM) 和改进的检索增强生成 (RAG) 管道，来增强可持续城市旅行的推荐系统。我们通过在提示增强阶段引入可持续性指标，确保推荐与可持续目标一致。实验结果显示，这种改进方法在大多数指标上均优于传统方法，证明了可持续性在 TRS 中的重要性。
>
> https://arxiv.org/abs/2409.18003
