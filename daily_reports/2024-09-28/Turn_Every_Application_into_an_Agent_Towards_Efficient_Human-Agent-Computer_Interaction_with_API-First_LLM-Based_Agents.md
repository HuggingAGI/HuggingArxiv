# 让每个应用都成为智能代理：探索基于 API 优先的 LLM 代理，实现更高效的人机交互
发布时间：2024年09月25日

`Agent应用`
> Turn Every Application into an Agent: Towards Efficient Human-Agent-Computer Interaction with API-First LLM-Based Agents
>
> 多模态大型语言模型 (MLLM) 让基于 LLM 的代理能够直接与应用界面 (UI) 互动，提升其在复杂任务中的表现。然而，这些代理常因频繁的 UI 交互而面临高延迟和低可靠性。为此，我们推出了 AXIS，一个优先通过 API 而非 UI 操作的 LLM 代理框架，并能自动探索应用以扩展 API。实验显示，AXIS 在 Office Word 上将任务完成时间缩短了 65%-70%，认知负荷降低了 38%-53%，且准确率保持在 97%-98%。我们的研究不仅开创了新的人-代理-计算机交互 (HACI) 框架和 UI 设计原则，还探索了将每个应用转化为代理的可能性，为未来的 Agent OS 奠定了基础。
>
