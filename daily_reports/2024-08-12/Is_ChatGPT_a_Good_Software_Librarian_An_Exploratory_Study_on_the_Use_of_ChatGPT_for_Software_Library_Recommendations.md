# ChatGPT 能否胜任软件图书管理员的角色？本研究探索其在软件库推荐方面的应用。
发布时间：2024年08月09日

`推荐系统`
> Is ChatGPT a Good Software Librarian? An Exploratory Study on the Use of ChatGPT for Software Library Recommendations
>
> 软件库对软件系统的功能、效率和可维护性至关重要。随着开发者日益依赖大型语言模型 (LLM) 来简化编码，这些模型推荐合适库的能力变得尤为关键，但这一领域仍有待深入探索。本文评估了 ChatGPT 作为软件图书馆员的表现，并指出了改进方向。我们通过 GPT-3.5 Turbo 为 10,000 个 Stack Overflow 问题生成 Python 代码，发现 ChatGPT 比人类开发者更频繁地使用第三方库，尤其是那些广泛且成熟的库。但推荐的库中有 14.2% 的许可证限制未被明确告知，6.5% 的库无法即开即用，可能导致开发者困惑和时间浪费。因此，ChatGPT 需提供更多关于可维护性和许可证的明确信息。我们建议开发者在集成 LLM 生成的代码前，加强依赖管理并仔细核查库的许可证。
>
> https://arxiv.org/abs/2408.05128
