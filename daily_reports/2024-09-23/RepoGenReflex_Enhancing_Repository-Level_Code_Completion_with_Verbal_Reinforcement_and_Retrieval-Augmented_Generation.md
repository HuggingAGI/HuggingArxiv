# RepoGenReflex：借助语言强化与检索增强生成，提升仓库级代码补全能力
发布时间：2024年09月19日

`RAG`
> RepoGenReflex: Enhancing Repository-Level Code Completion with Verbal Reinforcement and Retrieval-Augmented Generation
>
> 在复杂的软件工程任务中，解决问题的关键在于理解和修改大型代码库中的多个组件。因此，在仓库层面，准确提取相关信息以实现高效的代码完成至关重要。尽管现有工具已取得一定进展，但在动态优化检索和生成过程方面仍显不足。为此，我们提出了 RepoGenReflex 框架，通过结合检索增强生成（RAG）与语言强化学习（VRL），实现仓库级别代码完成的动态优化，确保每次都能提供最佳结果。
>
> https://arxiv.org/abs/2409.13122
