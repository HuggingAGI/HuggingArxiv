# HR-Agent：专为 HR 应用打造的任务导向对话 LLM 代理
发布时间：2024年10月14日

`Agent应用`
> HR-Agent: A Task-Oriented Dialogue (TOD) LLM Agent Tailored for HR Applications
>
> LLM 的最新进展虽已惠及教育、金融等领域，但在 HR 领域，仍有数百个重复性流程（如访问请求、医疗索赔和休假申请）未被触及。我们借鉴 LLM 代理在写作辅助和客户支持等任务上的成功经验，推出了 HR-Agent。这是一个高效、保密且专为 HR 设计的对话系统，旨在自动化处理医疗索赔和访问请求等重复性任务。值得一提的是，在系统运行过程中，对话数据不会被发送到 LLM，从而确保了 HR 任务所需的保密性。
>
> https://arxiv.org/abs/2410.11239
