# 大型语言模型在软件工程中的应用：全面调查
发布时间：2024年09月04日

`代码编写`
> Large Language Model-Based Agents for Software Engineering: A Survey
>
> 大型语言模型 (LLM) 的进步催生了基于 LLM 的 AI 代理，这些代理通过感知和利用外部资源，显著提升了 LLM 的多功能性和专业性。在软件工程 (SE) 领域，基于 LLM 的代理已展现出卓越成效。多个代理与人类互动的协同作用，为解决复杂 SE 问题带来了新希望。我们在此提供了一份关于 SE 领域基于 LLM 代理的全面调查，涵盖 106 篇论文，并从 SE 和代理视角进行分类。同时，我们也探讨了该领域的挑战与未来方向。调查仓库详见 https://github.com/FudanSELab/Agent4SE-Paper-List。
>
> https://arxiv.org/abs/2409.02977
