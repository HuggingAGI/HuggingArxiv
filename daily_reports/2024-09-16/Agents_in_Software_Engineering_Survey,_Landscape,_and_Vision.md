# 软件工程中的代理：全面调查、现状分析与未来展望
发布时间：2024年09月13日

`代码编写`
> Agents in Software Engineering: Survey, Landscape, and Vision
>
> 近年来，LLM 在软件工程领域取得了显著成就，广泛应用于各类任务。许多研究已将 LLM 与 SE 结合，采用代理概念。然而，现有工作的发展背景、优化任务的方式及 SE 中 LLM 代理框架的梳理尚不深入。本文首次调查了这些研究，提出包含感知、记忆和行动三大模块的 LLM 代理框架，并总结当前挑战，展望未来机遇。相关论文已整理至 GitHub 仓库：https://github.com/DeepSoftwareAnalytics/Awesome-Agent4SE。
>
