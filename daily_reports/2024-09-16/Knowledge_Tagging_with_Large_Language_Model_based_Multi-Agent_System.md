# 大型语言模型驱动的多智能体系统知识标签
发布时间：2024年09月12日

`Agent应用`
> Knowledge Tagging with Large Language Model based Multi-Agent System
>
> 在现代智能教育应用中，知识标签对于问题至关重要，涵盖学习进度诊断、练习题推荐和课程内容组织。传统上，这些标签由教育专家完成，因其需要深刻的语义理解和逻辑联系能力。随着NLP技术的发展，如预训练语言模型和LLM，研究者开始探索自动化知识标签。本文探讨了多代理系统在处理复杂知识定义和数值约束方面的优势，并通过MathKnowCT数据集展示了其卓越性能。这凸显了LLM多代理系统在克服先前方法挑战中的潜力。最后，深入讨论了自动化知识标签的影响，强调了在教育领域应用LLM算法的积极成果。
>
> https://arxiv.org/abs/2409.08406
