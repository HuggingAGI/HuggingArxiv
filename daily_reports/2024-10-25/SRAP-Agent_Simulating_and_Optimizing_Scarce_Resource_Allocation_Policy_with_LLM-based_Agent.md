# SRAP-Agent：借助 LLM 代理，模拟并优化稀缺资源的分配策略
发布时间：2024年10月17日

`Agent应用`
> SRAP-Agent: Simulating and Optimizing Scarce Resource Allocation Policy with LLM-based Agent
>
> 公共稀缺资源分配在经济学中至关重要，直接影响社会效率与公平。传统研究因理想化假设和数据限制而受限。我们创新提出SRAP-Agent框架，融合LLM于经济模拟，弥合理论与现实的鸿沟。以公共住房分配为例，我们通过广泛模拟实验验证其可行性，并采用优化算法。源码见https://github.com/jijiarui-cather/SRAPAgent_Framework。
>
> https://arxiv.org/abs/2410.14152
