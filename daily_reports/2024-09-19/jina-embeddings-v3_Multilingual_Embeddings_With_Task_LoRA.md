# jina-embeddings-v3：融合任务 LoRA 的多语言嵌入
发布时间：2024年09月16日


> jina-embeddings-v3: Multilingual Embeddings With Task LoRA
>
> 摘要：我们推出了 jina-embeddings-v3，一款拥有 5.7 亿参数的尖端文本嵌入模型，在多语言和长上下文检索任务中表现卓越，支持长达 8192 个 token 的上下文。该模型配备了任务专属的低秩适应（LoRA）适配器，确保在查询-文档检索、聚类、分类和文本匹配中生成高质量嵌入。此外，Matryoshka 表示学习技术融入训练过程，使得嵌入维度截断灵活且不影响性能。在 MTEB 基准测试中，jina-embeddings-v3 在英语任务上超越了 OpenAI 和 Cohere 的最新嵌入模型，并在所有多语言任务中表现优于 multilingual-e5-large-instruct。
>
