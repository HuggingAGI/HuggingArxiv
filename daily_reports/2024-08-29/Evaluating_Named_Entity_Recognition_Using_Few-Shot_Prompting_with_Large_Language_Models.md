# 通过少样本提示，利用大型语言模型评估命名实体识别。
发布时间：2024年08月28日

`提示工程`
> Evaluating Named Entity Recognition Using Few-Shot Prompting with Large Language Models
>
> 本文探讨了大型语言模型在命名实体识别 (NER) 任务中应用 Few-Shot Prompting 的效果。传统 NER 系统依赖大量标注数据，成本高昂且耗时。而 Few-Shot Prompting 或 in-context learning 则使模型能通过极少示例识别实体。我们评估了 GPT-4 等先进模型在 NER 任务中的 few-shot 性能，并发现尽管与完全监督基准存在差距，但这些模型在适应新实体类型和领域时表现优异，仅需少量数据。此外，我们还研究了提示工程、输出格式引导及上下文长度对性能的影响。本研究凸显了 Few-Shot Learning 在减少对大型标注数据集依赖、提升 NER 可扩展性和可访问性方面的潜力。
>
> https://arxiv.org/abs/2408.15796
